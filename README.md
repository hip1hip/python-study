# python-study

# 가상환경 생성
uv venv

# 패키지 설치(필요한 경우)
uv add pandas

# VS Code 에서 가상환경 연결하기
1. Ctrl + Shift + p 를 눌러서 Python: Select Interpreter 검색
2. 리스트 중에서 ./venv/Scripts/python.exe 또는 **Star (Recommeneded)**라고 표시된 항목을 선택합니다.
3. 이렇게 하면 터미널을 새로 열 때마다 자동으로 가상환경이 활성화((.venv) 표시가 뜸)됩니다.
